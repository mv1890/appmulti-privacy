Politica de privacidad de AppMulti

Fecha de entrada en vigor: 28 de marzo de 2026
Ultima actualizacion: 28 de marzo de 2026

AppMulti (en adelante, la App) se compromete a proteger la privacidad de las personas usuarias. Esta politica explica que datos se recogen, como se usan, con quien se comparten y que derechos tienes.

1. Responsable del tratamiento

Responsable: Marc Vergara (proyecto AppMulti)
Correo de contacto: marc1890@gmail.com

2. Datos que tratamos

Segun el uso que hagas de la App, podemos tratar:

  • Datos de cuenta e identificacion: correo electronico de acceso e identificador de usuario.
  • Datos que introduces voluntariamente: notas, entrenamientos, habitos, menus y otra informacion de productividad.
  • Archivos y contenido multimedia: imagenes o archivos que subas.
  • Datos tecnicos minimos para seguridad, diagnostico y estabilidad.
  • Datos de salud (si habilitas funciones relacionadas), con permisos explicitos.

3. Como obtenemos los datos

  • Directamente de ti, cuando creas cuenta o introduces informacion.
  • De servicios tecnicos necesarios para autenticacion y funcionamiento.
  • Del sistema del dispositivo cuando autorizas permisos concretos.

4. Finalidades del tratamiento

  • Prestar las funcionalidades de la App.
  • Guardar y sincronizar informacion entre dispositivos.
  • Permitir funciones colaborativas cuando las uses.
  • Mantener seguridad, prevenir abuso y resolver incidencias.
  • Mejorar rendimiento, estabilidad y experiencia de uso.
  • Cumplir obligaciones legales cuando proceda.

5. Base juridica

  • Ejecucion del servicio solicitado.
  • Consentimiento para funciones opcionales.
  • Interes legitimo en seguridad e integridad del servicio.
  • Cumplimiento de obligaciones legales.

6. Permisos y datos sensibles

Los permisos del dispositivo se usan solo para la funcionalidad correspondiente y conforme a esta politica. Puedes revocarlos desde los ajustes del sistema.

7. Comparticion de datos

No vendemos datos personales. Podemos compartir datos con proveedores que actuan por cuenta del responsable para prestar el servicio (por ejemplo, infraestructura de base de datos, autenticacion y almacenamiento).

8. Publicidad

Si la App muestra anuncios, pueden usarse SDK de publicidad para mostrar y medir anuncios. La presencia de anuncios se declara en Google Play cuando corresponda.

9. Conservacion de datos

Conservamos los datos mientras mantengas una cuenta activa o sea necesario para prestarte el servicio, salvo solicitud de eliminacion u obligacion legal.

10. Seguridad

Aplicamos medidas tecnicas y organizativas razonables para proteger los datos frente a acceso no autorizado, perdida o alteracion.

11. Transferencias internacionales

Si se realizan transferencias internacionales para prestar el servicio, se aplicaran garantias adecuadas conforme a normativa vigente.

12. Menores

La App no esta dirigida a menores que no cuenten con autorizacion o supervision legal exigible.

13. Tus derechos

Puedes ejercer derechos de acceso, rectificacion, supresion, oposicion, limitacion, portabilidad y retirada del consentimiento. Para ello, escribe a marc1890@gmail.com.

14. Cambios en esta politica

Podemos actualizar esta politica para reflejar cambios legales o funcionales de la App. Publicaremos la version actualizada indicando fecha.
